Devuelve un árbol binario que tiene una Struct como nodo.

Se crea un árbol binario que contiene Structs como nodos. Esto permite almacenar la información de las rutas. Recorrido pre-orden e in-orden. Lo importante es que la ruta se almacena dentro de una lista.

Precondition
cierto
Postcondition
Se devuelve un árbol binario además de una lista que contiene las ciudades que forman parte de la ruta.
Coste\n Lineal: O(n)
