Se devuelve el exceso de un producto que tiene la ciudad. 

El exceso determina si la ciudad quiere comprar o vender el producto\+: si el exceso es positivo, la ciudad tiene más unidades que las necesarias, y puede vender el producto. En cambio, si el exceso es negativo, el parámetro implícito contiene menos unidades que las necesarias\+: necesita comprar el producto. \begin{DoxyPrecond}{Precondition}
